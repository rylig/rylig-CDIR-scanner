# 🚀 اسکنر CDIR ریلگ (Rylig)

![تصویر اسکنر Rylig CDIR](https://github.com/rylig/rylig-CDIR-scanner/blob/main/screenshot.jpg?raw=true)

یک اسکنر سریع، بهینه و چندنخی برای پینگ گرفتن از IPها که با پایتون نوشته شده است. این ابزار با تمرکز بر سادگی، عملکرد بالا و کاربردی بودن در تحلیل شبکه طراحی شده است.

---

## ✨ ویژگی‌ها

* **اسکن چندنخی (Multi-threaded)** – اسکن سریع بازه‌های بزرگ IP با امکان تنظیم تعداد تردها
* **پشتیبانی از CIDR** – امکان اسکن کامل ساب‌نت‌ها (مثل `/24`)
* **ذخیره خودکار** – IPهای فعال به‌صورت خودکار در فایل `active.txt` ذخیره می‌شوند

---

## 🛠 نصب

### 1. کلون کردن ریپازیتوری

```bash
git clone https://github.com/rylig/rylig-CDIR-scanner.git
cd rylig-CDIR-scanner
```

### 2. نصب پیش‌نیازها

```bash
pip install -r requirements.txt
```

---

## ⚙️ نحوه استفاده

### 1. آماده‌سازی بازه‌های IP

یک فایل (مثلاً `range.txt`) بسازید و بازه‌های IP را داخل آن قرار دهید.

هر خط باید شامل یک رنج CIDR باشد:

```
8.6.112.0/24
8.6.144.0/24
8.6.145.0/24
```

---

### 2. اجرای اسکنر

اسکریپت را اجرا کنید:

```bash
python scanner.py
```

سپس از شما خواسته می‌شود:

* مسیر فایل رنج‌ها را وارد کنید (مثلاً `range.txt`)
* تعداد تردهای همزمان را مشخص کنید (تعداد IPهایی که همزمان اسکن می‌شوند)

---

### 3. مشاهده نتایج

* تمام IPهای پاسخ‌گو به‌صورت خودکار در فایل زیر ذخیره می‌شوند:

```
active.txt
```

---

## 📌 نکات

* تعداد ترد بیشتر = سرعت بالاتر، اما فشار بیشتر روی شبکه
* فقط روی شبکه‌هایی که مالک آن هستید یا مجوز دارید استفاده کنید
* بهترین عملکرد روی اتصال‌های پایدار شبکه حاصل می‌شود
